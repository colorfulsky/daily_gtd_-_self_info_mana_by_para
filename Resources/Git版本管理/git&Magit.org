** 提交远程仓库
*** git-bash操作
1) 设置ssh
#+begin_quote
ssh-keygen -t rsa -C "account-email-address"
#+end_quote
然后复制src_{rsa_id.pub}内容至github,ssh许可列表
2) 第一次上传
#+begin_src shell
  git push -u origin master 
#+end_src
